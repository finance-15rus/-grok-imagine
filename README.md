# Grok Imagine

AI image and video generation app powered by Hyperbolic (free tier) and Replicate.

## Features
- Text-to-image generation
- Text-to-video generation (via Replicate)
- Multiple aspect ratios and styles
- Generation history with IndexedDB storage
- Toast notifications

## Setup

1. Get free API key at [hyperbolic.xyz](https://hyperbolic.xyz) (1000 credits/month)
2. Open `index.html` in browser
3. Configure API key in Settings

## Deploy

Cloudflare Pages: Import this repo in Cloudflare Dashboard → Pages

## Tech Stack
- Vanilla JS SPA
- Cloudflare Pages
- Hyperbolic / Replicate APIs
